
# AI-Powered Real-Time NIDS & Analyst Dashboard 🛡️

## 📌 Project Overview
A high-performance, real-time Network Intrusion Detection System (NIDS) designed to classify known cyber threats and detect "Zero-Day" anomalies using the **CIC-IDS2017** dataset. The system provides transparent model insights via **Explainable AI (XAI)** and visualizes live traffic through a modern **React Dashboard**.
